一个简单的键盘按键测试工具。

## ✨介绍

使用 Bun、TypeScript、Tailwind CSS、shadcn/ui 构建的键盘按键测试工具。

## 🚀功能

- 实时检测、高亮键盘按键
- 输入预览

## 📦安装与使用

```bash

# 克隆项目
git clone --depth 1 https://github.com/hnchengzong/Keyboard-Tester.git

# 进入项目目录
cd ./Keyboard-Tester

# 安装依赖
bun install

# 启动
bun dev

# 或者bun run dev

# 访问 http://localhost:3000

```
